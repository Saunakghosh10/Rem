## 🎛️ Requirement
- [x] Node js v 16+
- [x] Java v 13 (for lavalink server)
- [x] yarn  `npm i -g yarn `
## ⚙️ Installation 
Fill all requirement in `src/config.js` or `.env` **(rename `.env.example` to `.env`)**

• [yarn](https://yarnpkg.com/)

For install all dependencies use
```
yarn
```
To start your bot 
```
yarn start
```
**Or**

• [Npm](https://www.npmjs.com/)

For install all dependencies use
```
npm install
```
To start your bot 
```
npm start
```
